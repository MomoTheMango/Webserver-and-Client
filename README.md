# :globe_with_meridians: 웹서버와 클라이언트
간단한 웹서버와 클라이언트입니다.

# 개요
> Boost/asio 라이브러리를 이용하여 웹서버와 클라이언트를 만들었습니다. 웹서버는 URL의 파라미터를 처리 및 사용하여 응답을 생성합니다. 클라이언트는 HTTP요청을 통해 외부 데이터(json)을 받아 출력합니다.
